This project

You probably know the html replit template. If this is the case, you can easily go to the next step and implement an API to improve your websites with Nodejs, a Javascript framework.
Indeed, passing from a completly static website to a dynamic one can be hard. That's why I've choosed to do that for the template jam.
Note:It's my first template, so please tell me in the comments what you think about it 😉
During the weeks in which I realized this template, I've tried to improve the following things:
  • Improve server's speed
    • Changing the file intitled package.json
    • Doing some changements in .replit files
    • Using read streams
  • Well organize the project
    • The pages, api endpoints and static files are placed in separeted folders
    • All the main files are in the manager folders, and they all do a specific task
  • Write about how the project works, describe and comment on each of the modules (it was the hardest part 🤔)
Here a list of all features:
  • Quick wake up call
  • Fast static and pages sending
  • Dynamic route segments
  • RegExp paths
  • Possibility to compress pages using gZip in a different directory (you can stille modify your files)
  • No framework
  • Possibility to create API endpoints
  • Peolple of all levels can use it: you can simply manage the HTML pages and static ressources, but also antirely personnalize the server
  • Some docs to know what the parts are for.
  • You can create an amazing website from this 🥳
